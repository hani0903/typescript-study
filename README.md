## 폴더 구조

```
typescript-study/
│
├── 01_inflearn/                # 강의별 정리본
│   ├── section1/
│   │   ├── 섹션 1. 강의 소개 및 타입스크립트 개론.md
│   │   ├── code/               # 실습 코드
│   │   └── images/             # 강의 이미지 자료
│   │   
│   └── ...
│
├── docs/                       # 공식 문서 정리 예정
│
├── README.md
└── ...

```

`01_inflearn`: 강의별로 `section`을 나누어 정리합니다.
- 각 `section` 폴더에는 다음 파일/폴더가 존재합니다.
  - `~.md`: 해당 섹션을 정리한 마크다운 문서
  - `code/`: 실습 코드
  - `images/`: 강의 자료 이미지
